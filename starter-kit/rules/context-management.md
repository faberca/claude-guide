# Context Management

## Output
- Terminal output: 3 lines max. Save the rest to MD files.
- Code/logs/implementation -> never print. Save to file if needed.
- Tables: 5 rows max. More than that -> save to file.
- Sub-agent results -> 1-2 line summary only.

## When Context Gets Full
1. 70% full: /compact focus on [current task]
2. 90%+ full: save handoff file -> continue in new session
   - Save to docs/handoff/handoff_YYYYMMDD_topic.md
   - Include: goal, done, not done, changed files, next steps
3. Important discovery: save to memory or project docs
